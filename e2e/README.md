# SaleADS Mi Negocio E2E (Playwright)

This folder contains an environment-agnostic end-to-end test for the full
"Mi Negocio" workflow:

- Google login
- Sidebar "Mi Negocio" expansion
- "Agregar Negocio" modal checks
- "Administrar Negocios" sections
- Legal pages ("Términos y Condiciones" / "Política de Privacidad")
- Final PASS/FAIL JSON report

## Why it is environment-agnostic

- No hardcoded SaleADS domain is used.
- It relies on either:
  - the browser already being on the login page, or
  - `SALEADS_LOGIN_URL` / `SALEADS_BASE_URL` environment variables.

## Install

```bash
cd /workspace/e2e
npm install
npm run install:browsers
```

## Run

Use one of these options:

```bash
# Option 1: provide environment URL dynamically
SALEADS_LOGIN_URL="https://<your-env-login>" npm run test:saleads-mi-negocio

# Option 2: headed run for interactive debugging
SALEADS_LOGIN_URL="https://<your-env-login>" npm run test:headed
```
