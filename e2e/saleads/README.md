# SaleADS Mi Negocio E2E

Playwright end-to-end test for the complete **Mi Negocio** workflow:

- Login with Google.
- Open **Mi Negocio** in sidebar.
- Validate **Agregar Negocio** modal.
- Open **Administrar Negocios** view and validate all account sections.
- Validate legal links (**Términos y Condiciones** and **Política de Privacidad**) including:
  - heading/content checks,
  - new-tab handling,
  - returning to the app tab,
  - final URL capture.
- Export a final PASS/FAIL report for all required validation blocks.

## Setup

```bash
cd /workspace/e2e/saleads
npm install
npx playwright install
```

## Run

If your environment requires a direct URL, provide it via `SALEADS_START_URL`:

```bash
SALEADS_START_URL="https://your-saleads-environment/login" npm run test:mi-negocio
```

If your runner already opens the login page automatically, run without URL:

```bash
npm run test:mi-negocio
```

Optional:

- `HEADLESS=false` to run headed mode.

## Evidence and report outputs

Artifacts are generated by Playwright under:

- `test-results/` (screenshots, trace/video on failure)
- `artifacts/test-results.json` (Playwright JSON report)
- `10-final-report.json` attachment per test run containing:
  - PASS/FAIL for:
    - Login
    - Mi Negocio menu
    - Agregar Negocio modal
    - Administrar Negocios view
    - Información General
    - Detalles de la Cuenta
    - Tus Negocios
    - Términos y Condiciones
    - Política de Privacidad
  - Final URLs for legal pages.
